You Have To Collect Gold($)And Avoid Obstacles 
Here are the commands to play
'a' to move to left
'd' to move to right
'w' to move up
's' to move down
'p' to pause and continue later
'b' to break out the game without saving
Note That In Case Using OS not Linux click Enter After Every Input
